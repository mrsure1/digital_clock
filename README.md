# 🕒 디지털시계 (Digital Clock) - 사이버펑크 에디션

**디지털시계**는 PC 바탕화면에 띄워 사용할 수 있는 프리미엄 사이버펑크 스타일의 투명 시계 애플리케이션입니다. 네온 효과와 글래스모피즘(Glassmorphism) 디자인으로 작업 환경을 더욱 멋지게 만들어 줍니다.

![시계 이미지 예시](https://via.placeholder.com/600x300/121212/00ffff?text=Cyber+Clock+Preview)

## ✨ 주요 기능

-   **미니멀 디자인**: 평상시에는 시간만 깔끔하게 표시됩니다.
-   **창 자유 조절**: 윈도우 창을 조절하듯 모서리를 드래그하여 시계 크기를 마음대로 조절할 수 있습니다.
-   **개인화 설정**:
    -   **색상 조절**: 네온 숫자의 색상을 슬라이더로 변경 가능.
    -   **밝기 조절**: 시계의 밝기를 환경에 맞게 조절 가능.
-   **스마트 정보**:
    -   **실시간 기온**: 설정한 지역의 현재 기온을 30분마다 업데이트하여 표시.
    -   **12시간제 표시**: 오전(AM)/오후(PM) 한글 표시기 제공.
-   **알람 기능**: 설정한 시간에 5초간 프리미엄 전자음이 울리며 팝업으로 알려줍니다. (팝업을 닫으면 소리가 멈춥니다.)
-   **백그라운드 실행**: 터미널 창 없이 시계만 깔끔하게 띄울 수 있는 전용 실행 방식 제공.

## 🚀 시작하기

### 🔧 1. 설치 방법 (일반 사용자용)
1.  이 저장소에서 `디지털시계.zip` 파일을 다운로드합니다.
2.  `C:\Program Files` 또는 원하는 위치에 압축을 풉니다.
3.  `디지털시계.exe` 파일을 실행합니다.
    -   *팁: 바탕화면에 바로가기를 만들어 사용하면 더욱 편리합니다.*

### 🛠️ 2. 개발자용 (소스 코드 실행)
Node.js와 npm이 설치되어 있어야 합니다.

```bash
# 저장소 클론
git clone https://github.com/mrsure1/digital_clock.git
cd digital_clock

# 종속성 설치
npm install

# 실행
npm start
```

## ⚙️ 설정 가이드

-   **시간 표시**: 시(Hour)가 한 자리일 경우 앞의 '0'이 보이지 않아 더욱 깔끔합니다.
-   **기온 설정**: 하단 '명령 설정'을 눌러 도시 이름(예: Seoul, Busan)을 입력하고 저장하세요.
-   **알람**: 24시간 형식(예: 14:30)으로 입력하여 실시간으로 관리할 수 있습니다.
